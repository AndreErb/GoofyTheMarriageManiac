Goofy The Marriage Maniac 
- Das Spiel zur Hochzeit -

Vers. 1.0.0.1

Installation:
1. setup.exe ausführen
2. Nur falls benötigt: "Microsoft .NET Framework 4.0" Installation bestätigen
3. Nur falls benötigt: "Microsoft XNA Framework" Installation bestätigen
4. "Goofy The Marriage Maniac" Installation bestätigen
5. Desktopverknüpfung wird erstellt und Spiel startet automatisch

Deinstallation:
Systemsteuerung => Programme und Funktionen 
=> Rechte Maust. auf "Goofy The Marriage Maniac" => Deinstallieren

Tasten:
Links gehen:  Pfeil Links  <- 
Rechts gehen: Pfeil Rechts -> 
Springen: Leertaste
Schießen: X
Vollbild an/aus: F12
